### RAG-Based AI Summarizer

* **Stack:** React, Node.js, Express, MongoDB, Google Gemini API / LangChain
* **Overview:**
An intelligent document processing and retrieval application that allows users to upload complex documents or text inputs and receive concise, context-aware AI summaries utilizing Retrieval-Augmented Generation (RAG).
* **Key Features:**
* Intelligent document chunking and vector-based context retrieval.
* Instant, context-aware text summarization using generative AI.
* Responsive, history-tracking dashboard to view past summaries.
* Secure API endpoint management for external data handling.


* **Technical Highlights:**
* Responsive design
* REST API integration
* Authentication & authorization
* Performance optimization

### GSoC Org Finder (Open Source Contributions)

* **Stack:** React, TypeScript, Tailwind CSS
* **Overview:**
A feature-rich web platform designed to help student developers easily navigate, filter, and find past Google Summer of Code (GSoC) organizations to streamline their open-source contribution journey.
* **Key Features:**
* **Real-time Search Statistics:** Integrated a dynamic insights panel highlighting total found organizations and dominant languages.
* **Performance-Driven Filtering:** Optimized continuous organization lookup sequences via custom debouncing implementations.
* **Accessible UI Elements:** Enhanced accessibility profiles across frameworks by correcting low-contrast text strings in dark mode layouts.
* **Advanced Admin Controls:** Granular filtering parameters based on issue difficulty and mentor availability.


* **Technical Highlights:**
* Modular architecture
* Production deployment
* Debounced front-end optimization
* Cross-theme accessibility validation
